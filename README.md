# damoa

macOS 메뉴바에 상주하는 공부 시간 추적 + 할 일 관리 앱

> 오늘 할 일은 오늘 끝내자

뽀모도로 타이머, 할 일 관리, 공부 기록을 하나로 합친 미니멀한 도구.
매일 빈 목록에서 시작하며, 미완료 항목은 자동으로 이월되지 않는다.

---

## 주요 기능

- **메뉴바 상주** — 타이머 실행 중에는 카운트다운, 평소에는 오늘 누적 시간을 메뉴바에 표시
- **할 일 관리** — 추가 / 완료 체크 / 삭제, 이름 변경(우클릭 컨텍스트 메뉴)
- **타이머** — 항목별로 5·10·15·20·25·30분 또는 직접 입력. 일시정지·재개·중단 지원. 중단 시 남은 시간 저장, 다음에 이어서 재개 가능
- **알림** — 타이머 완료 시 시스템 알림
- **날짜별 기록** — 하루 총 누적 시간, 항목별 누적 시간, 어제 대비 증감 표시
- **연속 일수** — 매일 기록이 쌓인 연속 일수 표시
